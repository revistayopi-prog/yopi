# Imágenes YOPI

Aquí van todas las fotografías del sitio.

## Estructura

```
images/
├── hero/           ← Imagen principal del hero (1600×900px mínimo)
├── fotoreportajes/ ← Galerías completas
├── biorelatos/     ← Fotos de investigaciones (800×500px)
├── colores-del-gris/
├── palabra-antigua/
├── palabra-florida/
└── biogramas/      ← Retratos de defensores y artistas
```

## Especificaciones técnicas

| Uso              | Tamaño mínimo  | Formato   | Peso máx |
|------------------|---------------|-----------|----------|
| Hero             | 1600×900px    | JPG/WebP  | 400KB    |
| Cards de notas   | 800×500px     | JPG/WebP  | 200KB    |
| Fotoreportajes   | 1200×800px    | JPG/WebP  | 300KB    |
| Biogramas        | 600×600px     | JPG/WebP  | 150KB    |

## Herramienta de compresión recomendada

[Squoosh](https://squoosh.app) — gratuito, sin subir a servidores externos.

## Convención de nombres

Usa guiones bajos, sin espacios ni acentos:  
`nahuizalco_mercado_01.jpg`  
`rio_lempa_comunidad.jpg`  
`juana_ama_retrato.jpg`
